# IronSaaS Cashflow

MicroSaaS para substituir planilhas de controle de saldo, lançamentos financeiros, DFC e projeção de caixa.

## Stack
- Frontend: Next.js
- Backend: FastAPI
- Banco: PostgreSQL

## Status
Base inicial do produto + Sprint A em andamento.

## Sprint A
- Auth base
- Empresas
- Contas
- Categorias
- Lançamentos CRUD
- Saldo por conta

## Estrutura
- `frontend/` interface web
- `backend/` API e domínio financeiro
- `docs/` documentação de produto e arquitetura
